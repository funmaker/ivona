# Ivona

Janusz V2 text to speech service.

## Disclaimer

- Files in `vox/Black Mesa Announcement` belong to Valve.
- Files in `vox/Space Station 13` belong to SS13 creators.

## Usage

### Run development

```bash
npm run start
```

### Build production

```bash
npm run build:prod
```

### Start production

```bash
npm run start:prod
```
